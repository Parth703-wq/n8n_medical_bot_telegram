# n8n_medical_bot_telegram
Medical assistant Telegram bot built with n8n



## Live Demo
https://t.me/medical_nidhi_bot

## Features
- Medical Q&A via text
- Voice input support
- PDF medical report analysis
- Voice response using TTS

## Tech Stack
- n8n
- Telegram Bot API
- OpenRouter (LLM)
- ElevenLabs (Speech)

## Setup
1. Import the JSON workflow into n8n
2. Create your own credentials:
   - Telegram Bot
   - OpenRouter API key
   - ElevenLabs API key
3. Attach credentials to nodes
4. Activate workflow

## Disclaimer
This bot is for informational purposes only. Always consult a doctor.
